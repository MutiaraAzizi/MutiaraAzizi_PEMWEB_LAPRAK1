1. Pertanyaan Seputar Tag

a. Apa yang dimaksud dengan tag pada HTML?

Jawaban :
Tag HTML adalah kode atau penanda yang digunakan untuk memberi instruksi kepada browser agar dapat menampilkan isi halaman web sesuai struktur yang diinginkan. Tag ditulis menggunakan tanda kurung siku, misalnya <p>, <h1>, <table>, dan lainnya. Setiap tag memiliki fungsi masing-masing sesuai kebutuhan pembuatan halaman web.

b. Apa itu tag pembuka dan penutup? Apakah terdapat pada semua tag?

Jawaban :
Sebagian besar tag HTML terdiri dari tag pembuka dan tag penutup. Tag pembuka digunakan untuk memulai suatu elemen, sedangkan tag penutup digunakan untuk mengakhiri elemen tersebut. Contohnya yaitu <p>Paragraf</p>.

Beberapa tag HTML tidak memiliki penutup karena hanya berfungsi sebagai elemen tunggal, seperti <br>, <hr>, <img>, dan <input>.

2. Pertanyaan Seputar Element

a. Apa yang dimaksud dengan element?

Jawaban :
Element HTML adalah keseluruhan bagian yang terdiri dari tag pembuka, isi konten, dan tag penutup. Contohnya <p>Belajar HTML</p> merupakan satu elemen paragraf.

b. Apa yang dimaksud tagname, attribute, dan content?

Jawaban :
Tagname adalah nama tag yang digunakan. Attribute adalah informasi tambahan pada tag pembuka seperti src, href, width, dan class. Content adalah isi yang berada di antara tag pembuka dan tag penutup.

c. Apakah element dapat ditumpuk?

Jawaban :
Ya, element dapat ditumpuk atau nested element. Contohnya yaitu <p><b>Teks Tebal</b></p>. Penulisannya harus rapi agar browser dapat membacanya dengan benar.

3. Pertanyaan Seputar Anchor

a. Apa yang dimaksud dengan anchor?

Jawaban :
Anchor adalah tag HTML yang digunakan untuk membuat tautan atau link. Tautan dapat menuju halaman lain, file tertentu, atau bagian tertentu pada halaman yang sama.

b. Bagaimana caranya anchor dapat berfungsi?

Jawaban :
Anchor bekerja menggunakan tag <a> dan atribut href yang berisi alamat tujuan. Anchor memudahkan navigasi pengguna saat membuka website.

c. Cara penggunaan anchor

Jawaban :
Contoh penggunaan anchor adalah: <a href="https://google.com">Google</a>. Saat diklik, pengguna akan diarahkan menuju halaman Google.

4. Pertanyaan Seputar Multimedia

A. Text

Soal: Sebutkan tag manipulasi dan struktur teks.

Jawaban :
Tag manipulasi teks seperti <b>, <i>, <u>, dan <mark>. Sedangkan tag struktur teks meliputi <h1> sampai <h6> serta <p>.

B. Image

Soal: Mengapa gambar tidak tampil dan muncul teks?

Jawaban :
Biasanya karena src salah, file tidak ditemukan, atau format tidak didukung. Jika muncul teks berarti browser menampilkan atribut alt.

C. Audio

Soal: Bagaimana memasukkan audio dari web?

Jawaban :
Gunakan tag <audio> dengan atribut src yang berisi link audio dari internet dan tambahkan controls.

D. Video

Soal: Jika autoplay tidak berjalan?

Jawaban :
Biasanya browser memblokir video yang memiliki suara otomatis. Tambahkan atribut muted agar autoplay berjalan.
